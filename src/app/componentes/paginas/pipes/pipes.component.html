Introducción

En Angular, los pipes son herramientas que permiten transformar visualmente datos en las plantillas de componentes. Pueden utilizarse para formatear texto, fechas, números y realizar otras operaciones de presentación. Los pipes se aplican en las plantillas mediante el símbolo | y son útiles para mejorar la legibilidad y presentación de la información en la interfaz de usuario.

Angular Pipes - Slice

El pipe slice en Angular se utiliza para extraer una porción de una cadena, array u otro tipo iterable. Permite especificar el índice de inicio y, opcionalmente, el índice de finalización para determinar el segmento que se debe extraer.

Ejemplo:

Supongamos que tenemos una cadena y queremos mostrar solo los primeros 5 caracteres en un componente Angular.

Texto: 'Angular es poderoso y flexible'

Resultado: {{ texto | slice:0:5 }}

Angular Pipes - Decimal

El pipe decimal en Angular se utiliza para formatear números con decimales. Permite especificar el número de dígitos enteros y decimales que se deben mostrar.

Ejemplo:

Supongamos que tenemos un número y queremos mostrarlo con dos decimales en un componente Angular.

Número: 1234.5678

Resultado: {{ numero | number:'1.2-2' }}

Angular Pipes - Percent

El pipe percent en Angular se utiliza para formatear números como porcentajes. Multiplica el valor por 100 y agrega el símbolo de porcentaje (%).

Ejemplo:

Supongamos que tenemos un número y queremos mostrarlo como un porcentaje en un componente Angular.

Número: 0.75

Resultado: {{ porcentaje | percent }}

Angular Pipes - Currency

El pipe currency en Angular se utiliza para formatear números como valores de moneda. Permite especificar el código de moneda y otras opciones de formato.

Ejemplo:

Supongamos que tenemos un número y queremos mostrarlo como un valor de moneda en un componente Angular.

Número: 1234.56

Resultado: {{ valorMoneda | currency:'USD':'symbol':'1.2-2' }}

Angular Pipes - Json

El pipe json en Angular se utiliza para convertir un objeto en su representación JSON. Es útil para depuración y visualización de datos en formato JSON.

Ejemplo:

Supongamos que tenemos un objeto y queremos mostrar su representación JSON en un componente Angular.

Objeto: {{ objeto | json }}

Angular Pipes - Async

El pipe async en Angular se utiliza para manejar observables directamente en las plantillas. Proporciona una manera fácil de suscribirse a observables y actualizar la interfaz de usuario cuando los datos cambian.

Ejemplo:

Supongamos que tenemos un observable y queremos mostrar su valor en un componente Angular.

Observable: {{ observableValue | async }}

Angular Pipes - Date

El pipe date en Angular se utiliza para formatear fechas según un formato específico. Permite personalizar la presentación de fechas en las plantillas de Angular.

Ejemplo:

Supongamos que tenemos una fecha y queremos mostrarla en un formato específico en un componente Angular.

Fecha: {{ fecha | date:'dd/MM/yyyy HH:mm:ss' }}

Angular Pipes - Register Locale Data

El método registerLocaleData en Angular se utiliza para registrar datos de localización específicos de un idioma. Permite habilitar el soporte de otros idiomas en tu aplicación.

Ejemplo:

Supongamos que queremos habilitar el soporte para el idioma francés en nuestra aplicación Angular.

Fecha en francés: {{ fecha | date:'medium' }}

Angular Pipes - TitleCase

El pipe titlecase en Angular se utiliza para capitalizar la primera letra de cada palabra en una cadena. Proporciona una manera fácil de presentar palabras en formato de título.

Ejemplo:

Supongamos que tenemos una cadena y queremos capitalizar la primera letra de cada palabra en un componente Angular.

Cadena: {{ cadena | titlecase }}

Angular Pipe Personalizado - EmojiFy

El siguiente ejemplo utiliza un pipe personalizado llamado emojiFy que convierte palabras clave en emojis.

Ejemplo:

Mensaje Original: {{ mensaje }}

Mensaje Emoji-ficado: {{ mensaje | emojiFy }}
