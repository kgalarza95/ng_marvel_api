Creando la estructura de nuestro proyecto

Comienza organizando tu proyecto Angular para una estructura eficiente y fácil de mantener. Utiliza el Angular CLI para generar componentes, módulos y servicios.

Instalando el Bootstrap/Material con Angular-CLI

Añade estilos atractivos a tu aplicación Angular utilizando Bootstrap o Material Design. El Angular CLI facilita la instalación de estas bibliotecas para mejorar la apariencia de tu aplicación.

Configurando el navbar y otros componentes

Personaliza el navbar y otros componentes de la interfaz de usuario para que se ajusten al diseño y la experiencia de usuario deseados. Aprende a integrar estos componentes en tu aplicación Angular.

Rutas en Angular

Explora el enrutamiento en Angular para navegar entre diferentes secciones de tu aplicación. Crea rutas para organizar y acceder fácilmente a las distintas partes de tu SPA.

RouterLink y RouterLinkActive en Angular

Utiliza las directivas RouterLink y RouterLinkActive para facilitar la navegación y resaltar visualmente la ruta activa en tu aplicación Angular.

Paso a Paso

  • Paso 1: Importa las directivas necesarias en tu componente o módulo.
  • Paso 2: Configura las rutas en tu aplicación Angular. Esto se hace generalmente en un archivo de rutas (`app-routing.module.ts`).
  • Paso 3: Utiliza `RouterLink` en tus enlaces para navegar entre rutas.
  • Paso 4: Añade estilos específicos para la clase activa con `RouterLinkActive`.
Cómo usar la API de Marvel

La API de Marvel proporciona acceso a una amplia variedad de información sobre personajes, cómics y más. Aquí hay algunos pasos básicos para comenzar a usar la API de Marvel sin necesidad de escribir código:

  • Paso 1: Regístrate en el sitio web de desarrolladores de Marvel para obtener tu clave de API.
  • Paso 2: Obtén tu clave pública y privada desde la sección de desarrolladores de Marvel.
  • Paso 3: Utiliza tu clave pública y privada para construir URLs de solicitud según la documentación de la API.
  • Paso 4: Realiza solicitudes HTTP a las URL de la API para obtener información sobre personajes, cómics, etc.
  • Paso 5: Procesa y muestra la información obtenida en tu aplicación según tus necesidades.

Recuerda revisar la documentación oficial de la API de Marvel para conocer los endpoints disponibles, parámetros de solicitud y cualquier otro detalle importante.
